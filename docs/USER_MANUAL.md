# Amazon Knowledge Support System - User Manual

## For Customers

### Getting Started

1. **Access the System**
   - Open web browser
   - Navigate to the application URL
   - You will see the login page

2. **Login**
   - Username: `user1`
   - Password: `user123`
   - Click "Login" button

### Raising a Query

1. Go to **"Raise Query"** tab
2. Fill in the form:
   - **Your Name**: Your username (pre-filled)
   - **Order ID**: Optional - Amazon order number
   - **Query Type**: Select from dropdown (Return Request, Replacement, etc.)
   - **Describe Your Issue**: Explain your problem in detail
3. Click **"Submit Query"**
4. Note your **Ticket ID** (e.g., TKT12345)

### Chatting with Support Agent

1. Go to **"Support Chat"** tab
2. Select your ticket from dropdown
3. Type your message in the text box
4. Click **"Send"**
5. Click **"Refresh Chat"** to see agent's reply
6. Continue conversation until issue is resolved

## For Support Agents

### Getting Started

1. **Login**
   - Username: `agent1`
   - Password: `agent123`

### Handling Tickets

1. Go to **"Ticket Queue"** tab
2. See **Open Tickets** section
3. Click on a ticket to view details
4. Click **"Handle Ticket"** to start working on it
5. Ticket moves to **"In Progress"**

### Searching Knowledge Base

1. Go to **"Knowledge Base Search"** tab
2. Enter customer's query
3. Click **"Search Solutions"**
4. Review top 5 relevant solutions
5. Use text area to copy solution content
6. Select all (Ctrl+A) and copy (Ctrl+C)

### Chatting with Customer

1. Go to **"Active Chats"** tab
2. Select ticket from dropdown
3. See conversation history
4. Type response in text box
5. Click **"Send Reply"**
6. Click **"Refresh Chat"** to see new messages

### Resolving Tickets

1. Go to **"Ticket Queue"** > **"In Progress"** section
2. Select your ticket
3. Fill **two separate feedbacks**:

   **A. Customer Resolution (Visible to Customer):**
   - Explain how issue was resolved
   - Example: "Your return has been processed. Refund in 7-10 days."

   **B. KB Feedback (For Developers Only):**
   - Rate KB helpfulness: Very Poor to Excellent
   - Provide improvement suggestions
   - Example: "KB search was accurate but slow"

4. Click **"Resolve Ticket"**

## Troubleshooting

### Cannot Login
- Check username and password
- Ensure caps lock is off
- Contact system administrator

### Ticket Not Showing
- Click "Refresh" button
- Check you're in correct tab
- Verify ticket was submitted successfully

### Chat Not Updating
- Click "Refresh Chat" button manually
- Check internet connection
- Reload page if needed

### Knowledge Base Search Not Working
- Verify knowledge base is built
- Check error message
- Contact technical support
